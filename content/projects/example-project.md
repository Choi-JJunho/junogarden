# junogarden 웹사이트

개인 실험 공간으로 사용하기 위한 웹사이트입니다.

## 기술 스택

- **프레임워크**: Next.js 15
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **콘텐츠**: Markdown

## 주요 기능

### 블로그 시스템
마크다운 파일로 블로그 글을 작성할 수 있습니다.

### 프로젝트 쇼케이스
진행한 프로젝트들을 정리하고 공유할 수 있습니다.

## 디자인 컨셉

- 미니멀한 디자인
- 초록색 계열 컬러
- Brick dot 패턴 배경
